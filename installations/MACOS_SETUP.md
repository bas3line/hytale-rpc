# macOS Setup Guide

## Download

1. Go to the [Releases page](https://github.com/bas3line/hytale-rpc/releases)
2. Download `HytaleRPC.dmg`

## Install

1. Double-click `HytaleRPC.dmg`
2. Drag the **HytaleRPC** app to your **Applications** folder
3. Eject the disk image

## Usage

1. Open **HytaleRPC** from Applications
2. Look for "H" icon in your menu bar (top right)
3. Launch Discord (required for RPC to work)
4. Launch Hytale
5. Your status will appear on Discord:
   - Look at the menu bar to see current status
   - Status displays: "Waiting for Hytale", "Connected", "In Main Menu", "Playing", etc.
   - Click the "H" icon to access menu options and quit

## What It Shows

- In Main Menu
- Loading World
- Playing Singleplayer (with world name)
- Playing Multiplayer (with server)
- Time elapsed

## Auto-Start

1. Open System Settings
2. Go to General → Login Items
3. Click the `+` button
4. Select **HytaleRPC** from Applications
5. Make sure it's enabled (toggle is on)

## Troubleshooting

### App won't open

If you get a security warning:
1. Open System Settings → Privacy & Security
2. Look for "HytaleRPC was blocked from opening"
3. Click **Open Anyway**

### Not showing on Discord

1. Make sure Discord is running
2. Make sure Hytale is running
3. Check the menu bar icon status
4. Try quitting and reopening the app

### Menu bar icon missing

1. Launch HytaleRPC from Applications
2. Look in the top-right menu bar for "H" icon
3. If prompted for permissions, allow them

## Uninstall

1. Quit HytaleRPC
2. Move to trash: `/Applications/HytaleRPC`
3. Remove from login items if set up
